# CodeAlpha C++ Programming Internship Projects

C++ projects built during the CodeAlpha 1-Month Programming Internship 2026.

## Task 1 - To-Do List Application

A console-based task manager application built in C++.

### Features
- Add tasks with custom categories
- Mark tasks as completed
- View pending and completed tasks separately
- Search tasks by category
- Delete tasks
- Auto-saves all tasks to tasks.txt using File I/O

### How to Run
g++ -o todo_list "Todo List.cpp"
todo_list.exe

### Concepts Used
- Object Oriented Programming (Classes)
- Vectors
- File I/O (ifstream, ofstream)
- Loops and Conditionals

## Task 2 - File Encryption & Decryption

A program that encrypts and decrypts any text file using two layers of security.

### Features
- Encrypt any .txt file
- Decrypt back to original message
- Double layer security (Caesar Cipher + XOR)
- Preview file contents before and after encryption

### How to Run
g++ -o file_encryption "File Encryption.cpp"
file_encryption.exe

### Algorithm Used
- Caesar Cipher: shifts letters by a chosen number (1-25)
- XOR layer: applies extra encryption on every character

### Concepts Used
- File Streams (ifstream, ofstream)
- Strings and Loops
- Object Oriented Programming (Classes)
- Basic Encryption Logic
